# KAYON-Integrated-Consults
This is the official website for **KAYON Integrated Consults**, based in Lokoja, Nigeria, a company providing professional construction and project management services in Lokoja. Specializes in residential builds and commercial projects.

## Features
* **Responsive Design:** Built with Tailwind CSS (works on mobile and desktop).
* **Contact Form:** Integrated with Formspree to send leads directly to email.
* **Hosted on:** GitHub Pages.

## Tech Stack
* HTML5
* Tailwind CSS

For project inquiries, please use the contact form on the website or visit us at our Lokoja office.
